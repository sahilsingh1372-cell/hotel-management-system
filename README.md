# Hotel Management System

## Project Overview
This project is a web-based application developed to manage hotel operations such as room booking, customer details, and billing in a simple and efficient manner.

---

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

---

## Features
- Room booking management
- Customer record handling
- User-friendly interface
- Secure data storage

## How to Run the Project
1. Download or clone the repository
2. Place the project folder in server directory (XAMPP/WAMP)
3. Import database file into MySQL
4. Open browser and run the project
